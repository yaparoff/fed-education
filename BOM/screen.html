Screen

  • availHeight - высота экрана пользователя для вывода информации (без панели задач браузера, полосы прокрутки), window.screen.availHeight
  • availWidth - ширина экрана пользователя для вывода информации (без панели задач браузера, полосы прокрутки), window.screen.availWidth
  • colorDepth - битовая глубина цветовой палитры, window.screen.colorDepth
  • height - общая высота экрана, window.screen.height
  • pixelDepth - глубина цвета в битах на пиксель, window.screen.pixelDepth
  • width - общая ширина экрана, window.screen.width
